PluginConfiguration

class PluginConfiguration(val identifier: String) : Configuration, PluginIdentifiable, PluginEntitiesCallable, PluginAfterTrackCallable, PluginFilterCallable

Configuration for a custom tracker plugin. Enables you to add closures to be called when and after events are tracked in the tracker.
